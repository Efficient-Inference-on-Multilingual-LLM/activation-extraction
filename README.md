# activation-extraction
Extracting the activations of each decoder layer in a decoder-only based LLMs.

## Setup
To install the libraries, use this command:
```
pip install -r requirements.txt
```
Tested using `python 3.12.11`. 

## How to Run
### Activation Extraction
- For topic classification task, run `scripts/run_tc.sh` script.
- For machine translation task, run `scripts/run_mt.sh` script.
